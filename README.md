# Visual Lab web crawler

## Script to crawl data from [Visual Lab](http://visual.ic.uff.br/dmi/)

### RUNNING:

1. Configure your settings in `settings.json`
2. Run in terminal:
   ```sh
   pip install -r requirements.txt
   ```
3. Run in terminal:
   ```sh
   python -m visual_lab
   ```
4. Wait until finish.
